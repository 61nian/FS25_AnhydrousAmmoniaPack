**Test Team:** *JundtFarm, Math, Tillman, QcFS*

I release this revised FS25 version because the original FS25 port by *Red’s Modding* had an issue that applicators could not be recolored.
I have fixed them, now support custom paintings.

**Important note:**  
This modified version is specially tailored for *TisonK*’s "FS25_SoilFertilizer".
The built-in filltype "Anhydrous" and its matching script files have been completely removed.
Since "FS25_SoilFertilizer" already includes the "Anhydrous" filltype registration, duplicate entries are no longer required.
